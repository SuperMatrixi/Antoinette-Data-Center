# Antoinette Data Center

AI and Web3.0 tech hub, incubator, and data center project in Paris (Bobigny-Cœur-de-Ville).

## Project Structure

```
Antoinette-Data-Center/
├── index.html
├── README.md
├── .gitignore
├── robots.txt
├── sitemap.xml
├── css/
│   └── style.css
├── js/
│   ├── main.js
│   └── chatbot.js
├── docs/
│   ├── content.md
│   └── mindmap.md
├── scripts/
│   ├── build.py
│   └── generate.py
├── assets/
│   ├── images/
│   ├── icons/
│   ├── logos/
│   └── videos/
└── .github/
    └── workflows/
        └── deploy.yml
```

## Where to Upload Media

- Put photos, banners, and backgrounds in `assets/images/`
- Put UI icons in `assets/icons/`
- Put brand logos in `assets/logos/`
- Put video files in `assets/videos/`

## Beginner Bulk Upload (Git)

```bash
git add .
git commit -m "Organize project structure and add assets"
git push origin main
```
